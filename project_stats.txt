PROJECT SAMENVATTING - TRAVIAN CLASSIC CLONE
Huidige Status
We hebben een volledig functionele Travian Classic clone gebouwd met:

Technologie Stack
Backend: PHP 8+ met PDO

Database: MySQL

Frontend: Vanilla HTML/CSS/JS (responsive, mobile-first)

Geen frameworks (geen Bootstrap, geen Laravel, puur vanilla)

Database Structuur (16+ tabellen)
users - gebruikers met tribe (roman/teuton/gaul)

villages - dorpen met x/y coördinaten

resources - resources met productie rates

buildings - 14 gebouwen met levels

building_queue - upgrade queue

troops - troepen per dorp

training_queue - training queue

troop_movements - aanvallen/ondersteuning

battles - battle logs

map_cells - grid kaart (-50 tot 50)

alliances + alliance_members + alliance_messages

messages - privé berichten

quests + user_quests

player_stats, server_settings, etc.

Gebouwen (14 stuks)
main_building - basis, verhoogt bouwsnelheid

woodcutter - hout productie (+5/level)

clay_pit - klei productie (+5/level)

iron_mine - ijzer productie (+5/level)

farm - graan productie (+5/level)

warehouse - opslag hout/klei/ijzer (+2000/level)

granary - opslag graan (+2000/level)

barracks - infanterie training (main_building >= 3)

stable - cavalerie training (main_building >= 5, barracks >= 3)

smithy - wapen/ pantser upgrades (main_building >= 3, barracks >= 1)

academy - technologieën (main_building >= 3, barracks >= 3)

marketplace - handel (main_building >= 3)

embassy - allianties (main_building >= 1)

wall - verdediging (main_building >= 1)

Troepen per tribe
Romans:

Legionnaire (infanterie)

Praetorian (verdediger)

Teutons:

Club Swinger (aanvaller)

Spearman (verdediger)

Gauls:

Phalanx (verdediger)

Swordsman (aanvaller)

Tribe Bonussen
Romans: 10% goedkopere gebouwen, 10% snellere bouw

Teutons: 20% goedkopere troepen

Gauls: 20% snellere troepen beweging, 5% snellere training

Werking
Cron jobs draaien elke minuut voor resources, queues en bewegingen

Resources worden elk uur geproduceerd (opslag beperkt)

Gebouwen hebben wachtrijen met timers

Troepen training heeft wachtrij met timers

Kaart is een grid (-50 tot 50) met dorpen, oases, lege velden

Aanvallen hebben reistijd gebaseerd op afstand

Berichten systeem werkt tussen spelers

Hoe het werkt met GitHub:
1. Maak een GitHub account aan (gratis)
Ga naar https://github.com en maak een account aan.

2. Maak een nieuwe repository aan
Klik op de groene knop "New repository"

Naam: bijv. travian-clone

Kies "Public" (dan kan ik het zien) of "Private" (dan moet je mij toevoegen als collaborator)

Klik "Create repository"

3. Upload je bestanden
Je kunt dit doen via:

Git command line (aanbevolen voor developers)

GitHub Desktop (makkelijker voor beginners)

Handmatig uploaden via de GitHub website (sleep bestanden erin)

4. Ik kan dan:
De hele repository bekijken via de URL

Alle bestanden doorlezen

De volledige code structuur zien

Specifieke bestanden bekijken

Zien wat de laatste versie is

5. Voorbeeld URL
Als je repository publiek is, ziet het er zo uit:

text
